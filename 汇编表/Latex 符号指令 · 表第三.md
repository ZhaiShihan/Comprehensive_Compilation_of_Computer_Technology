### 壹  关系符号

| 关系   | 符号             | 指令           |
| ---- | -------------- | ------------ |
| 大于等于 | $\geq$         | \geq         |
| 小于等于 | $\leq$         | \leq         |
| 大于   | $\textgreater$ | \textgreater |
| 小于   | $\textless$    | \textless    |
| 属于   | $\in$          | \in          |
| 恒等于  | $\equiv$       | \equiv       |


### 贰  上下标记

| 格式    | 表示                | 指令              |
| ----- | ----------------- | --------------- |
| 上角标   | $x^2$             | x^2             |
| 下角标   | $a_2$             | a_2             |
| 上下均标  | $\Sigma_{i=1}^n$  | \Sigma_{i=1}^n  |
| 平均值   | $\overline{x}$    | \overline{x}    |
| 下划线   | $\underline{x}$   | \underline{x}   |
| 下划连接线 | $a\underline{~}b$ | a\underline{~}b |
